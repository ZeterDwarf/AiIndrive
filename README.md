# AgroScoring AI: MVP для Decentrathon 5.0

## Описание проекта
AgroScoring AI — это прототип Data-driven системы скоринга сельхозпроизводителей для получения государственных субсидий. 
Этот проект решает **Кейс 2** хакатона Decentrathon 5.0 (AI for Government).

Вместо устаревшего подхода "кто первый подал заявку, тот и получил", система применяет методы машинного обучения для ранжирования заявителей на основе их исторических показателей, эффективности и потенциала развития (Merit-based подход).

## Ключевые возможности MVP (Gate 1)
- **Scoring Engine:** Модель машинного обучения (на данный момент Random Forest), оценивающая заявителей по ряду параметров: урожайности, истории субсидий, опыту и задолженностям. Возвращает score в диапазоне 0-100.
- **Explainability (Объяснимость):** Критическое требование хакатона. Система не только выдает балл, но и предоставляет массив текстовых причин — *почему* данный балл был присвоен, чтобы комиссия могла принять обоснованное решение. AI выступает только в роли ассистента, а не заменяет эксперта.

## Состав репозитория на первом этапе
- `architecture.md` — описание высокоуровневой архитектуры системы для финальной стадии.
- `main.py` — минимально работающий элемент (MVP). Модуль для обучения модели (пока на сгенерированных mock-данных) и функция скоринга `score_application()`.
- `requirements.txt` — зависимости проекта.

## Установка и запуск

1. Клонируйте репозиторий.
2. Создайте виртуальное окружение: `python -m venv .venv`
3. Активируйте окружение и установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
4. Запустите скрипт проверки:
   ```bash
   python main.py
   ```
Вы увидите результат скоринга для тестовой заявки и объяснение модели.
